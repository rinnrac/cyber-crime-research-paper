AHFMD introduces a unified detection framework that integrates:

- Real-time process monitoring using eBPF
- Memory artifact extraction with Volatility
- LLM-based feature explanation
- MITRE ATT&CK correlated threat classification

Real-Time Process Surveillance
- eBPF-based syscall monitoring
- Detection of suspicious process behavior
- Early identification of malicious activities

Memory Forensics
- Automated memory snapshot collection
- Volatility-based artifact extraction
- Detection of reflective DLL injection and memory-resident threats

AI-Powered Threat Explanation
- Fine-tuned Llama model
- Human-readable security explanations
- Improved analyst understanding and response
